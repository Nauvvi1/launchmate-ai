# LaunchMate AI 🚀

🌐 https://launchmate-ai-omega.vercel.app/  
🎥 https://www.youtube.com/watch?v=e1Q4eWEnzBQ  

## What it is

LaunchMate AI checks if your MVP is actually ready to be shown.

You give it:
- Demo URL
- GitHub repo
- basic product info

It gives you back:
- a readiness score
- clear issues
- and what to fix next

## The problem

Most MVPs fail not because of code, but because:

- landing page looks unfinished
- no documentation
- unclear business value
- no launch structure

So even good projects look "not ready".

## What this does better

Unlike typical hackathon tools:

- not just AI guessing
- not just static checklist

It combines:
✔ real technical checks (URL, page, metadata, docs)  
✔ structured scoring system  
✔ AI explanation layer (optional)

## Output

You get:

- Marketplace Readiness Score
- Checks (what is good / bad)
- Next steps (what to fix first)
- AI summary (short, practical, no fluff)

## Why it exists

Because building a MVP is not enough.

You need to know if it's actually ready to launch.

LaunchMate AI answers exactly that.
